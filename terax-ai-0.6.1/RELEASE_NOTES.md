## 📦 Installation Instructions
### macOS
Download the `.dmg` file and drag Terax to your Applications folder. If the app is blocked by Apple, run `xattr -cr /Applications/Terax.app` in your terminal.
### Windows
Run the `.msi` installer. Requires Windows 10 or 11.
### Linux
Download the `.AppImage`, make it executable (`chmod +x Terax.AppImage`), and run it.
